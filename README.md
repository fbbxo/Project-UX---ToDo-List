# 📝 My List - Todo App (Mid-Fidelity)

Projeto de um gerenciador de tarefas inteligente, focado em organização diária e análise de produtividade, desenvolvido como parte do curso de Sistemas para Internet.

---

## 👥 Público-alvo
Estudantes, profissionais e entusiastas de produtividade que buscam uma ferramenta simples, porém eficaz, para gerenciar metas diárias, visualizar o progresso semanal/mensal e priorizar atividades importantes.

## ✨ Funcionalidades
- **Sistema de Autenticação**: Proteção de dados com criação de conta e login (via LocalStorage).
- **Gerenciamento de Tasks (CRUD)**: Criar, ler, editar e excluir tarefas com título e descrição.
- **Filtro Diário Inteligente**: A tela principal exibe apenas as obrigações do dia corrente.
- **Priorização e Destaque**: Definição de níveis de prioridade (Alta, Média, Baixa) e marcação de favoritos (Estrela).
- **Calendário Interativo**: Filtre e visualize tarefas de qualquer data específica.
- **Analytics Real**: Gráficos dinâmicos que calculam a performance (feito vs. não feito) com base na semana ou no mês atual.
- **Persistência de Dados**: Os dados permanecem salvos no navegador mesmo após fechar a aba.

## 📱 Telas Planejadas
1.  **Login**: Entrada no sistema com link decorativo de recuperação de senha.
2.  **Sign Up**: Cadastro de novos usuários.
3.  **My List (Home)**: Visão focada nas tarefas de "Hoje".
4.  **Task Details (Modal)**: Interface de criação/edição com campos de data, hora, prioridade e estrela.
5.  **Analytics**: Gráfico de pizza e barras de progresso comparativas.
6.  **Calendar**: Histórico e planejamento de datas futuras.

## 🛠️ Tecnologias Utilizadas
- **HTML5**: Estruturação semântica da aplicação.
- **Tailwind CSS**: Estilização moderna, responsiva e baseada em utilitários.
- **JavaScript (ES6+)**: Lógica de manipulação de DOM, cálculos de datas e gerenciamento de estado.
- **LocalStorage API**: Armazenamento local de usuários e tarefas.
- **FontAwesome**: Biblioteca de ícones para interface intuitiva.
